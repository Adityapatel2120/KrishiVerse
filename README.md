# 🌾 KrishiVerse

An AI-Powered Multilingual Smart Farm Management & Crop Disease Detection Platform.

## Features

- 🌿 AI Crop Disease Detection
- 🚜 Farm Management
- 🌱 Crop Management
- 💰 Expense Tracking
- 📊 Dashboard & Analytics
- 🌐 Multilingual Support (English, Hindi, Gujarati)

## Tech Stack

### Frontend
- React 19
- Vite
- Tailwind CSS
- React Router
- Axios

### Backend
- Node.js
- Express.js
- MongoDB Atlas

### AI
- FastAPI
- TensorFlow
- OpenCV
